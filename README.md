# 🦷 Portfólio Profissional - Dra. Júlia Orzanqui Simões

Este projeto consiste no desenvolvimento de um site institucional (one page) para apresentação profissional de uma cirurgiã-dentista em início de carreira.

## 🎯 Objetivo
Criar uma presença digital moderna, elegante e funcional, com foco em transmitir confiança, acolhimento e facilitar o contato com pacientes.

## 💻 Tecnologias utilizadas
- HTML5
- CSS3

## 🎨 Design e conceito
O layout foi desenvolvido com base em uma identidade visual minimalista, utilizando tons neutros (bege e branco) para transmitir:
- leveza
- sofisticação
- acolhimento

O design prioriza a experiência do usuário (UX), com navegação simples e conteúdo organizado em uma única página.

## 📐 Estrutura do projeto
O site foi desenvolvido no formato one page, contendo as seguintes seções:

- Home (apresentação principal)
- Sobre
- Serviços
- Diferenciais
- Depoimentos
- Contato

## 📱 Responsividade
O projeto foi estruturado para se adaptar a diferentes tamanhos de tela, proporcionando uma boa experiência tanto em dispositivos móveis quanto em desktops.

## 🔗 Funcionalidades
- Botão de contato direto via WhatsApp
- Links para redes sociais
- Navegação fluida entre seções
